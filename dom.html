    • etiqueta head:

  • interactuar con nodos y elementos

    • nodos de tipo element: es una etiuqeta html en particular

      • capturar un elemento por el nombre de una etiqueta:
        document.getElementsByTagName( " li " )

      • caputar un elemento por el slector de class:
        document.getElementsByClassName( " " )

      • capturar un elemento por selector id:
        document.getElementById( " " )

      • capturar un elemento por el atributo Name: formularios
        document.getElementsByName( " " )

      • capturar por un selector cualquiera: captura solo el primer elemento
        document.querySelector( " " )

      • capturar por un selector cualquiera: captura todos los elementos
        document.querySelectorAll( " " )


    • nodos de tipo tex: son los textos que tienen adentro una etiqueta

  • accendiendo a tributos y Data-Attributes de html

    • 1 con la nomenclatura del punto:
      const atributo_lang= document.documentElement.lang;

    • 2 con la nomenclatura del punto:
      const atributo_con_nomenclatura_del_punto= document.getElementById( "" ).nombre_del_atributo;

    • 3 con el metodo getAttribute:
      const atributo_con_getAttribute= document.getElementById( "" );
      atributo_con_getAttribute.getAttribute("nombre del atributo");

  • modificando los atributos

    • 1.1 atributo_lang=en;

    • 2.1 atributo_con_nomenclatura_del_punto=valor_acambiar

    • 3.1 atributo_con_getAttribute.setAttribute("attibuto", "valor_acambiar" )

  • interactuando con estilos css

    • capturar en una variable un estilo css

      const stilo= document.getElementById("ul_partes_dom");

    • capturar en una variable una propiedad en concreto de css

      const color=getComputedStyle(stilo).getPropertyValue("color");

    • modificar un estilos css

      stilo.style.backgroundColor="red"
      color=stilo.style.color="yellow"

    • con variables css

      let color_body=getComputedStyle(body).getPropertyValue("--color_fondo");
      body.style.backgroundColor=color_body;

    • saber si un elemento tiene una class:

      const classes=document.querySelector(".titulo1")
      classes.classList.contains("titulo1")

    • ageregar una class

      const add_clsase=document.querySelector(".titulo1")
      add_clsase.classList.add("titulo2")

    • remover una class

      add_clsase.classList.remove("titulo1")

    • el toogle si tiene una clase se la quita y si no se la pone

      add_clsase.classList.toggle("titulo2")

    • replace reemplazar una clase por otra

      add_clsase.classList.replace("titulo1","titulo2")
